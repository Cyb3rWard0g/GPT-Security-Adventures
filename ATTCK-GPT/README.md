# ATT&CK GPT Experiment
This experiment demonstrates how to use LangChain and OpenAI embeddings to index ATT&CK groups and query them using natural language.

# Requirements

* openai-0.27.4
* tiktoken-0.3.3
* scikit-learn-1.2.2
* tenacity-8.2.2
* matplotlib
* seaborn
* tqdm
* sqlalchemy-2.0.10
* unstructured-0.6.1
* python-dotenv-1.0.0
* faiss-cpu-1.7.4
* LangChain
